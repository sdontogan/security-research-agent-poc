# Acceptable use

This repository demonstrates defensive, read-only security research. Use it only for
domains and data you are permitted to investigate.

The project must not be used to support unauthorized scanning, exploitation, credential
attacks, persistence, evasion, malware delivery, destructive actions, or data
exfiltration. Its reports are informational and should be reviewed by a person before
they affect an operational security decision.

Public intelligence services have their own licenses, terms, quotas, and data-handling
rules. Users are responsible for following those requirements.
